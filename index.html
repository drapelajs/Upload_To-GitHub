First, save your html that you created with any additional files that go with it.

Second, right click in the folder where there is no files so it is just white. A window should appear and you want to click the "Git Bash" botton. A command script should pop up on your screen.

Third, type this command: $ git init

Fourth, type this command: $ git checkout -b gh-pages

Fifth, type this command: $ git status

Sixth, your files you want to add should be red. Now you want to type this: $ git add (filename). Example: $ git add index.html add this will add your html file.

Seventh, you want to type: $ git status. The files you added will be green.

Eighth, type: $ git commit -m"(with a message of what you did inside the quatation marks)".

After commiting the page(s), type: $ git status one more time. The text should be white.

Now you need to go to Github.com and sign in. you need to click the + sign in the top right hand corner. This will add a repository. Select the folder with the files you want on the internet. Then click enter.

Type this into your html file using brackets: $ git push -u origin gh-pages.
